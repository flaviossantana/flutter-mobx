# Flutter com MobX

Projeto com exemplo de integração com MobX:

## Getting Started

Este projeto foi adicionado as seguintes dependências:

#### dependencies:
* path_provider,
* hive,
* dio,
* flutter_mobx,
* mobx,
* flutter_modular

#### dev_dependencies:
* mobx_codegen,
* build_runner

#### scripts:
* build: flutter pub run build_runner watch
* clean: flutter pu run build_runner clean
